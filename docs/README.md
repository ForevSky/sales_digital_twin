## 项目概述

**销售数字分身** 面向 B2B 销售团队，通过 Agent 框架自动处理模拟的销售通话文本，完成：

1. **信息提取** — 客户名称、沟通摘要、需求痛点、下一步行动
2. **结构化输出** — 固定格式文本，可直接复制到 CRM
3. **CRM 更新建议** — 推断需变更的字段（状态、商机金额等）
4. **非销售场景兜底** — 输出「非销售场景，跳过处理」
5. **商业合同变更建议** — 识别价格、条款等变更需求
